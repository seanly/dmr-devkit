> Prerequisite: [01-overview.md](01-overview.md), [02-devkit.md](02-devkit.md)  
> Related: [06-tape.md](06-tape.md) for tape isolation in A2A.

## What is A2A

A2A (Agent2Agent) is a JSON-RPC protocol for inter-agent communication. `dmr-devkit` implements A2A via the `a2aserver` package, built on `a2a-go` v2.

Key features:
- **Agent Card**: Well-known endpoint for service discovery
- **SendMessage**: JSON-RPC method for task invocation
- **Streaming**: SSE-based progressive output delivery
- **Interoperability**: Works with dmr's A2A plugin and other `a2a-go` implementations

> **Note**: This is the community `a2a-go` v2 protocol, not Google ADK's built-in A2A. They are not compatible.

## Architecture

```
┌─────────────┐      HTTP POST      ┌─────────────────┐
│   Client    │ ──────────────────▶ │   http.ServeMux │
│  (dmr A2A   │  /a2a/invoke       │                 │
│   plugin)   │                     │  a2aserver.Register()
└─────────────┘                     │                 │
                                    │  ┌───────────┐  │
                                    │  │Agent Card │  │  GET /.well-known/agent.json
                                    │  └───────────┘  │
                                    │  ┌───────────┐  │
                                    │  │  Invoke   │  │  POST /invoke
                                    │  │  Handler  │  │
                                    │  └─────┬─────┘  │
                                    └────────┼────────┘
                                             │
                                             ▼
                                      ┌──────────────┐
                                      │ agent.Run()  │
                                      │ (per Task)   │
                                      └──────────────┘
```

## Registration

```go
package main

import (
    "net/http"
    "github.com/seanly/dmr-devkit/a2aserver"
    "github.com/seanly/dmr-devkit/devkit"
)

func main() {
    ctx := context.Background()
    kit, _ := devkit.Build(ctx, devkit.EnvOptions())
    defer kit.Close(ctx)

    mux := http.NewServeMux()
    a2aserver.Register(mux, a2aserver.Options{
        AgentName:       "MyAgent",
        Description:     "A helpful assistant",
        PublicInvokeURL: "https://example.com/a2a/invoke",
        MountPath:       "/a2a/invoke",
        TapeMode:        a2aserver.TapeModeAuto,
    }, kit.Agent)

    http.ListenAndServe(":8080", mux)
}
```

## Options

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `AgentName` | `string` | required | Agent display name |
| `Description` | `string` | required | Agent description |
| `PublicInvokeURL` | `string` | required | Absolute URL clients use for JSON-RPC POST |
| `MountPath` | `string` | `/invoke` | Path on local mux |
| `TapeMode` | `TapeMode` | `TapeModeAuto` | `Auto` = tape per TaskID; `Fixed` = constant tape |
| `TapePrefix` | `string` | `"a2a"` | Prefix for auto-generated tape names |
| `TapeName` | `string` | `"default"` | Constant tape name (Fixed mode only) |
| `ChunkSize` | `int` | `0` | SSE chunk size (0 = non-streaming) |
| `DefaultInputModes` | `[]string` | `[]` | Supported input modes |
| `DefaultOutputModes` | `[]string` | `[]` | Supported output modes |

### TapeMode

| Mode | Behavior | Use Case |
|------|----------|----------|
| `TapeModeAuto` | Each A2A Task gets its own tape (`prefix_taskID`) | Multi-user, stateless server |
| `TapeModeFixed` | All tasks share one tape | Single-user, persistent session |

## Agent Card

The Agent Card is served at `/.well-known/agent.json`:

```json
{
  "name": "MyAgent",
  "description": "A helpful assistant",
  "url": "https://example.com/a2a/invoke",
  "version": "1.0.0",
  "capabilities": {
    "streaming": true,
    "pushNotifications": false
  },
  "defaultInputModes": ["text"],
  "defaultOutputModes": ["text"]
}
```

Clients discover the agent by fetching this endpoint.

## SendMessage

The JSON-RPC endpoint handles `SendMessage`:

```json
{
  "jsonrpc": "2.0",
  "method": "SendMessage",
  "params": {
    "taskId": "task-123",
    "message": {
      "role": "user",
      "parts": [{"type": "text", "text": "Hello"}]
    }
  },
  "id": 1
}
```

Response:

```json
{
  "jsonrpc": "2.0",
  "result": {
    "taskId": "task-123",
    "status": "completed",
    "artifacts": [{
      "parts": [{"type": "text", "text": "Hello! How can I help?"}]
    }]
  },
  "id": 1
}
```

## Streaming (SSE)

When `ChunkSize > 0`, output is delivered as SSE events:

```go
opts := a2aserver.Options{
    ChunkSize: 100,  // Deliver output in 100-char chunks
}
```

Events:
- `TaskStatusUpdateEvent`: Status changes (working, input-required, completed)
- `TaskArtifactUpdateEvent`: Output chunks

## Environment Variables

For deployments behind NAT or TLS termination:

```bash
# Set the public-facing URL
export A2A_PUBLIC_INVOKE_URL=https://mydomain.com/a2a/invoke
```

The server uses this when `PublicInvokeURL` is not explicitly set in Options.

## Complete Example

See `examples/a2a_devkit_server/main.go`:

```go
package main

import (
    "context"
    "fmt"
    "log"
    "net/http"
    "os"

    "github.com/seanly/dmr-devkit/a2aserver"
    "github.com/seanly/dmr-devkit/devkit"
    "github.com/seanly/dmr-devkit/tool"
)

func main() {
    ctx := context.Background()
    opts := devkit.EnvOptions()
    if opts.APIKey == "" || opts.Model == "" {
        log.Fatal("AI_API_KEY and AI_MODEL required")
    }

    opts.Tools = []*tool.Tool{/* ... */}
    kit, err := devkit.Build(ctx, opts)
    if err != nil {
        log.Fatal(err)
    }
    defer kit.Close(ctx)

    publicURL := os.Getenv("A2A_PUBLIC_INVOKE_URL")
    if publicURL == "" {
        publicURL = "http://localhost:8080/a2a/invoke"
    }

    mux := http.NewServeMux()
    a2aserver.Register(mux, a2aserver.Options{
        AgentName:       "DevkitAgent",
        Description:     "A devkit-powered A2A agent",
        PublicInvokeURL: publicURL,
        MountPath:       "/a2a/invoke",
        TapeMode:        a2aserver.TapeModeAuto,
        TapePrefix:      "a2a",
    }, kit.Agent)

    port := os.Getenv("PORT")
    if port == "" {
        port = "8080"
    }
    fmt.Printf("A2A server listening on :%s\n", port)
    fmt.Printf("Agent card: %s/.well-known/agent.json\n", publicURL)
    log.Fatal(http.ListenAndServe(":"+port, mux))
}
```

## Client Interoperability

### From dmr CLI

```toml
[[plugins]]
name = "a2a"
[plugins.config]
base_url = "https://example.com/a2a/invoke"
```

### From Any a2a-go Client

```go
client := a2a.NewClient("https://example.com/a2a/invoke")
task, err := client.SendMessage(ctx, "Hello")
```
